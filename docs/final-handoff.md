## How to Use Instructions

### Prerequisites
- Web browser (Chrome, Firefox, Safari, or Edge)
- No external dependencies required
- No build process needed

### Getting Started

1. **Clone or download the project**:
   ```bash
   git clone <repository-url>
   cd project-pulse-dashboard
   ```

2. **Open the application**:
   - **Option A**: Navigate to the project folder and open `app/index.html` in your browser
   - **Option B**: Use a local web server:
     ```bash
     # Using Python 3
     python -m http.server 8000
     
     # Using Python 2
     python -m SimpleHTTPServer 8000
     
     # Using Node.js
     npx http-server
     ```
   - **Option C**: Use VSCode Launch (see below)

3. **Launch Using VSCode**:
   - Open the project in Visual Studio Code
   - The launch configuration "Run Project Pulse Dashboard" is configured in `.vscode/launch.json`
   - Press `F5` or select the launch configuration from the debug panel
   - The dashboard will open in your default browser

### Key Features

- **Dashboard Overview**: View project metrics and team statistics
- **Project Timeline**: Track milestones and deadlines
- **Task Management**: Monitor task progress and status
- **Team Collaboration**: Access team member information
- **Resource Allocation**: See resource distribution
- **Responsive Interface**: Works on all device sizes

### Data Management

- Project data is stored in `app/project-data.json`
- To update project information, edit the JSON file
- The dashboard automatically reflects data changes
- Maintain the JSON structure for proper functionality

### Customization

- **Styling**: Modify `app/styles.css` for visual changes
- **Content**: Update `app/index.html` for structural changes
- **Data**: Edit `app/project-data.json` for project information
- **Branding**: Update CSS variables for quick theme changes
